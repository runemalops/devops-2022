# DevOps 2022
A DevOps practice repository containing various tools and configurations for learning and implementing DevOps practices.
## Repository Structure
```
devops-2022/
├── Ansible/          # Ansible playbooks and configurations
├── Docker/          # Docker-related files and examples
├── DockerSwarm/      # Docker Swarm orchestration configs
├── LabSetup/         # Lab environment setup files
├── Vagrantfile.erb   # Vagrant template for VM provisioning
├── clean.sh          # Cleanup script
├── setup.sh          # Environment setup script
├── hosts             # Inventory hosts file
├── .gitignore
└── LICENSE
```
## Contents
### Ansible
Automation playbooks and configurations for infrastructure management.
### Docker
Container-related files including Dockerfiles and compose configurations.
### DockerSwarm
Docker Swarm mode configurations for container orchestration.
### LabSetup
Files for setting up lab environments for DevOps practice.
## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/runemalops/devops-2022.git
   cd devops-2022
   ```
2. Run the setup script:
   ```bash
   ./setup.sh
   ```
## Requirements
- Vagrant
- Docker
- Ansible
